About Me

I'm a passionate Frontend Developer who loves combining modern web technologies with thoughtful design, and brings brands to life online in a way that is both engaging and functional.

My Journey

My journey into frontend development started with curiosity about how websites work. After 13+ years in IT, helping organizations optimize datacenters and operations, I realized I wanted to understand how technology is experienced by end users.

What began as tinkering with HTML and CSS has grown into a full skill set that allows me to build beautiful, responsive, and functional websites.

I’m passionate about writing clean, maintainable code and designing interfaces that are intuitive and accessible. Whether using modern website builders or coding from scratch, I focus on simplicity and performance — no unnecessary “fluff”.
